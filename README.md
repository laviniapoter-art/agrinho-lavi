# 🌱 AgroTech Sustentável — Agrinho 2026

Este projeto foi desenvolvido especialmente para o **Concurso Agrinho 2026**. Trata-se de uma Landing Page responsiva e interativa criada para conscientizar a comunidade escolar e a sociedade sobre a importância das práticas sustentáveis na pecuária moderna.

O objetivo principal é mostrar como a tecnologia, o manejo correto do solo e o respeito ao bem-estar animal podem caminhar juntos para garantir uma produção de alimentos eficiente e amiga do meio ambiente.

---

## 🚀 Funcionalidades do Site

- **Design Moderno e Responsivo:** O site se adapta perfeitamente a computadores, notebooks, tablets e smartphones.
- **Seção Educativa:** Explicações claras sobre conceitos fundamentais como a técnica **ILPF** (Integração Lavoura-Pecuária-Floresta), o manejo rotacionado e a apresentação de soluções como comedouros sustentáveis.
- **Quiz Interativo:** Um jogo de perguntas e respostas dinâmico via JavaScript para testar os conhecimentos dos visitantes.
- **Experiência do Usuário (UX):** Menu de navegação fixo com efeito dinâmico no cabeçalho, efeitos de animação ao rolar a página e rolagem suave entre as seções.

---

## 🛠️ Tecnologias Utilizadas

Para a construção deste portal, utilizamos tecnologias web puras (Front-End padrão):

- **HTML5:** Para a estruturação semântica de todo o conteúdo.
- **CSS3:** Para a criação do layout moderno, paleta de cores voltada à natureza e responsividade.
- **JavaScript (ES6):** Para as animações de rolagem e a lógica de checagem do quiz interativo.
